## Installations and Overview
> At first You have to install python3 on your system
  
> Now write these commands :
- ` git install https://github.com/InferiorAK/GoponFIle ` <br>
- ` cd GoponFile `
  > For Linux
  
- ` ./mi_gopon.py -h `
  > For Windows
- ` python3 mi_gopon.py -h `
